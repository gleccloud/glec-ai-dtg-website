## 🚀 실행 방법

### 로컬 서버 시작
```bash
# 방법 1: npm script 사용
npm start          # http://localhost:3000
npm run dev        # http://localhost:3001 (자동 열기)

# 방법 2: 직접 실행
npx http-server . -p 3000 --cors
python3 -m http.server 8000
```

### 현재 실행 중인 서버
- **URL**: http://127.0.0.1:8082
- **네트워크**: http://172.30.1.19:8082
